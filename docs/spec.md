# プロジェクト仕様書：TATEYAMA BASE 北条 公式サイト

## 1. プロジェクト概要
千葉県館山市北条にある貸別荘「TATEYAMA BASE 北条」の公式Webサイト。
単なる宿泊予約サイトではなく、館山の魅力やアクティビティ（ゴルフ、SUP等）と連動した体験型宿泊施設としてのブランディングを行う。

### コンセプト
- 「アクアライン渋滞を回避し、館山を遊び尽くすためのベースキャンプ」
- 清潔感、開放感、そしてワクワク感を感じるモダンなデザイン。

## 2. ターゲットユーザー
- ゴルフ、SUP、釣り、麻雀などを楽しむアクティブなグループ（4〜8名）
- 渋滞を避けて前泊・後泊を検討する旅行者
- 北条海岸の夕日や富士山やその他館山観光を目的とする観光客・ファミリー
- 千葉周辺の大学生グループ

## 3. 技術スタック（推奨）
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- UI Components: shadcn/ui
- Icons: Lucide React

## 6. 実装上の詳細・追記が必要な項目

### 6.1. ビジュアル・デザイン
- メインビジュアル（Hero Section）の演出:
	- 没入型（フルスクリーン）：ブラウザいっぱいに夕日の写真を広げ、その上に白抜きのシンプルなロゴを置く。高級感と非日常感が出ます。
- カラーパレット（館山の「色」を定義する）:「ビーチハウス・モダン」パレット
	- メイン：清潔感のある白（#FFFFFF）と、館山の明るい海の色（#0077BE）などの寒色
	- アクセント：砂浜をイメージしたベージュ（#F5F5DC）
- フォントと質感（書体の選び方）
	- 日本語フォント: ゴシック体（モダン）
	- 質感: 可能であればグラモーフィズムを実装してみてほしい。

### 6.2. コンテンツ詳細
	- 規定に則って、Google Maps APIを使って、周辺の主要スポットにピンを立てたカスタムマップを作り、貸別荘周辺のおすすめスポットをまとめて下さい。
	- 掲載リスト案
		- コンビニ: セブン-イレブン 館山北条店（徒歩◯分）
		- スーパー: おどや 北条店（地元の新鮮な魚が買える！）
		- 飲食店: 繁華街の「館山駅前エリア」まで徒歩圏内。
		- 癒やし: 近くのマッサージ店や銭湯情報。
